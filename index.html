Скачать проект с gitgub.com

https://github.com/dezvell/realtimechat.git

В проекте уже предустановлен Yii2 вместе с node-socket extension

\vendor\yii-node-socket

Создаем базу данных. Dump базы лежит в корне приложения

/yii2basic.sql

Для работы чата нужно запустить server c помощью bash команды в корне приложения:

> chat

Если проект не индексируется по адресу /index.php нужно зайти в /web/index.php, в этом случае в файле /views/site/chat.php нужно поправить пути:

<script src="/js/server/node_modules/socket.io/node_modules/socket.io-client/dist/socket.io.js"></script>
<script src="/js/client/client.js"></script>

to

<script src="/web/js/server/node_modules/socket.io/node_modules/socket.io-client/dist/socket.io.js"></script>
<script src="/web/js/client/client.js"></script>
